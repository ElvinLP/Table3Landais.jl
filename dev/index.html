Table3Landais

Documentation for Table3Landais.

The package Table3Landais was designed to replicate using Julia Table n°3 from Camille Landais' paper "Assessing the Welfare Effects of Unemployment Benefits Using the Regression Kink Design".

Before using this package, please make sure you have downloaded the necessary files from the online Replication Package on the AER website.
